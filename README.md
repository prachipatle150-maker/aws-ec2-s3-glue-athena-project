# aws-ec2-s3-glue-athena-project
Healthcare Appointment Analytics Dashboard
Project Overview
This project focuses on analyzing medical appointment no‑show patterns to understand patient behavior and improve clinic efficiency. The interactive Power BI dashboard provides actionable insights into appointment attendance trends across age groups, gender, and time, helping healthcare providers reduce no‑shows and optimize scheduling.

 Key Objectives
Identify factors contributing to appointment no‑shows
Analyze patient attendance behavior by age group and gender
Monitor appointment trends over time
Support data‑driven decisions to improve healthcare operations

 Dashboard Highlights
Total Appointments: 111K
No‑Shows: 22K
No‑Show Rate: 20.19%
Average Waiting Days: 9.18

Key Insights
Young adults (20–35) and senior patients (56+) show higher no‑show rates
Patients without SMS reminders are more likely to miss appointments
Female patients account for a higher proportion of appointments
Peak appointment volume observed during mid‑period trends

Business Impact
